# Запуск сервера
	python matrixMultiplication.py [имя файла с матрицами] [i] [j] [k] [m]

i - кол-во строк в первой матрице   
j - кол-во столбцов в первой матрице   
k - кол-во строк во второй матрице   
m - кол-во столбцов во второй матрице

i - кол-во строк в первой матрице
